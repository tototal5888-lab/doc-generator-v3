# 🚀 快速開始指南

## 第一次使用

### 步驟 1: 安裝依賴
雙擊運行 `run_v3.bat`,腳本會自動:
1. 檢查 Python 環境
2. 安裝必要的依賴套件
3. 創建必要的目錄
4. 啟動應用程式

### 步驟 2: 配置 API
1. 打開瀏覽器訪問 http://localhost:5000
2. 點擊 "⚙️ API設定" 標籤
3. 選擇 AI 引擎 (推薦使用 Gemini)
4. 輸入對應的 API Key
5. 點擊保存

**獲取 API Key:**
- Gemini: https://makersuite.google.com/app/apikey
- OpenAI: https://platform.openai.com/api-keys

### 步驟 3: 上傳模板
1. 點擊 "📁 模板管理" 標籤
2. 拖放或點擊上傳您的模板文件
3. 支援格式: PDF, DOCX, PPT, TXT

### 步驟 4: 生成文檔
1. 點擊 "📝 生成文檔" 標籤
2. 選擇文檔類型
3. 選擇模板
4. 輸入需求描述
5. **選擇輸出格式** (這是 V3.0 的新功能!)
   - 📄 Word (.docx) - 可編輯文檔
   - 📊 PowerPoint (.pptx) - 簡報格式
   - 📕 PDF (.pdf) - 固定格式
   - 📝 Markdown (.md) - 純文本
6. 點擊 "✨ 生成文檔"
7. 下載生成的文檔

## 使用技巧

### 技巧 1: 選擇合適的格式
根據用途選擇格式:
- 內部使用 → Word
- 對外展示 → PowerPoint 或 PDF
- 技術文檔 → Markdown
- 長期保存 → PDF

### 技巧 2: 模板準備
好的模板能產生更好的結果:
- 結構清晰
- 格式統一
- 內容完整

### 技巧 3: 需求描述
詳細的需求描述能獲得更好的結果:
- 說明具體場景
- 列出關鍵要點
- 提供參考信息

### 技巧 4: 多格式生成
同一內容可以生成多種格式:
1. 生成 Word 版本用於編輯
2. 生成 PDF 版本用於分發
3. 生成 Markdown 版本用於存檔

## 常見問題

**Q: 如何更換 API?**
A: 在 "API設定" 中選擇不同的引擎並輸入對應的 Key。

**Q: 生成失敗怎麼辦?**
A: 檢查 API Key 是否正確,網絡是否正常。

**Q: 支援哪些模板格式?**
A: PDF, DOCX, DOC, PPTX, PPT, TXT。

**Q: 生成的文檔在哪裡?**
A: 在 "📚 文檔記錄" 中查看並下載。

**Q: 可以刪除舊模板嗎?**
A: 可以,在 "模板管理" 中點擊刪除按鈕。
