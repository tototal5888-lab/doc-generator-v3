- Python 3.10 或更高版本

### 依賴套件
所有依賴已列在 `requirements_v3.txt` 中

## 🚀 快速開始

### 1. 安裝依賴
```bash
pip install -r requirements_v3.txt
```

### 2. 配置 API 密鑰
在應用程式的 "API設定" 頁面中輸入您的 API 密鑰:
- **Google Gemini API** (推薦) - 取得密鑰: https://makersuite.google.com/app/apikey
- **OpenAI API** - 取得密鑰: https://platform.openai.com/api-keys

### 3. 啟動應用
```bash
python app_v3.py
```

### 4. 訪問應用
打開瀏覽器訪問: http://localhost:5000

## 📖 使用指南

### 生成文檔流程

1. **上傳模板**
   - 切換到 "模板管理" 標籤
   - 拖放或點擊上傳模板文件
   - 支援格式: PDF, DOCX, DOC, PPTX, PPT, TXT

2. **配置生成參數**
   - 切換到 "生成文檔" 標籤
   - 選擇文檔類型 (系統文檔/SOP/技術報告)
   - 選擇模板文件
   - 輸入需求描述
   - **選擇輸出格式** (DOCX/PPTX/PDF/Markdown)

3. **生成文檔**
   - 點擊 "生成文檔" 按鈕
   - 等待 AI 生成完成
   - 預覽生成結果
   - 下載文檔

4. **管理文檔**
   - 在 "文檔記錄" 中查看所有生成的文檔
   - 支援多格式文檔的下載

## 🎨 輸出格式說明

### Word 文檔 (.docx)
- **特點**: 可編輯、格式豐富
- **適用場景**: 
  - 需要後續編輯的文檔
  - 正式報告、提案
  - 內部文檔

### PowerPoint 簡報 (.pptx)
- **特點**: 結構化展示
- **適用場景**: 
  - 培訓材料
  - 項目展示
  - 會議簡報

### PDF 文檔 (.pdf)
- **特點**: 固定格式、通用性強
- **適用場景**: 
  - 對外分發
  - 長期歸檔
  - 打印文檔

### Markdown (.md)
- **特點**: 純文本、易於版本控制
- **適用場景**: 
  - 技術文檔
  - Git 倉庫文檔
  - Wiki 內容

### AI 引擎

  - 支援多種模型

## 🆕 V3.0 新功能詳解

### 1. 格式轉換引擎
新增 `FormatConverter` 類別,支援將 Markdown 轉換為:
- DOCX: 完整樣式支援,標題層級,列表
- PPTX: 自動分頁,標題頁,內容頁
- PDF: 專業排版,頁眉頁腳
- Markdown: 原生格式,附加元信息

### 2. 智能格式選擇
根據文檔類型自動建議最佳格式:
- 系統文檔 → 推薦 DOCX 或 PDF
- SOP → 推薦 DOCX
- 技術報告 → 推薦 PDF 或 Markdown

### 3. 格式預覽
生成後顯示格式圖標和類型,一目了然

### 4. 批量下載
支援下載不同格式的同一文檔內容

## 🎯 使用建議

### 選擇最佳格式

**Word (.docx)**
- ✅ 需要編輯內容
- ✅ 需要添加註釋
- ✅ 需要追蹤修訂
- ❌ 需要固定格式

**PowerPoint (.pptx)**
- ✅ 需要投影展示
- ✅ 結構化展示
- ✅ 培訓材料
- ❌ 長篇文檔

**PDF (.pdf)**
- ✅ 對外分發
- ✅ 固定格式
- ✅ 打印輸出
- ❌ 需要編輯

**Markdown (.md)**
- ✅ 技術文檔
- ✅ 版本控制
- ✅ Wiki/博客
- ❌ 需要富格式

## 🐛 常見問題

### Q1: 生成 PDF 時中文顯示問題?
A: 已使用 ReportLab 內建的中文支援,無需額外配置。

### Q2: PowerPoint 格式不理想?
A: PowerPoint 生成採用標準模板,可生成後手動調整。

### Q3: 支援自定義樣式嗎?
A: 目前使用預設樣式,未來版本會增加樣式自定義。

### Q4: 可以同時生成多種格式嗎?
A: 目前需要分別生成,可連續操作。
